# مكتبة الأصالة - Library Bookstore

موقع إلكتروني متكامل لبيع الكتب أونلاين، مبني بـ ASP.NET Core MVC، بتصميم كلاسيكي مستوحى من المكتبات القديمة.

## المميزات

### جانب العميل
- تصفح الكتب مع بحث وفلترة حسب القسم والسعر
- عرض تفاصيل كل كتاب
- سلة مشتريات (إضافة / تعديل الكمية / حذف)
- إتمام الطلب (Checkout)
- تسجيل حساب وتسجيل دخول
- تتبع حالة الطلبات (قيد التجهيز / تم الشحن / تم التسليم)

### لوحة تحكم الأدمن
- Dashboard بإحصائيات عامة (عدد الكتب، الطلبات، العملاء)
- إدارة الكتب (إضافة / تعديل / حذف / رفع صورة الغلاف)
- إدارة الأقسام (تصنيفات الكتب)
- إدارة الطلبات وتغيير حالتها

## التقنيات المستخدمة

- ASP.NET Core MVC
- Entity Framework Core (SQLite)
- ASP.NET Core Identity (تسجيل الدخول والأدوار)
- Bootstrap 5
- خطوط Cinzel + Tajawal

## متطلبات التشغيل

قبل التشغيل، تأكدي إن الأدوات دي متثبتة على الجهاز:

- [.NET SDK 8.0](https://dotnet.microsoft.com/download) أو أحدث
- Visual Studio 2022 أو VS Code (مع C# Dev Kit Extension)

للتأكد من التثبيت، افتحي Terminal واكتبي:

```bash
dotnet --version
```

## خطوات التشغيل

### 1. تثبيت الحزم المطلوبة (Packages)

افتحي Terminal داخل مجلد المشروع واكتبي:

```bash
dotnet restore
```

### 2. تثبيت أداة Entity Framework (لو مش مثبتة من قبل)

```bash
dotnet tool install --global dotnet-ef
```

### 3. إنشاء قاعدة البيانات

```bash
dotnet ef database update
```

هذا الأمر هينشئ قاعدة بيانات SQLite جديدة، ويضيف تلقائيًا:
- الأقسام الأساسية (روايات، علمي، أطفال، ديني، تاريخ)
- حساب أدمن افتراضي

### 4. تشغيل المشروع

```bash
dotnet run
```

أو في VS Code / Visual Studio: اضغطي **F5**

هيظهر لينك في الـ Terminal شبه:

```
Now listening on: https://localhost:5001
```

افتحيه في المتصفح.

## بيانات دخول الأدمن الافتراضية

| الحقل | القيمة |
|---|---|
| البريد الإلكتروني | admin@library.com |
| كلمة المرور | Admin@123 |

⚠️ **مهم:** يُفضل تغيير كلمة مرور الأدمن بعد أول تسجيل دخول لأسباب أمنية.

للوصول للوحة التحكم بعد تسجيل الدخول، اذهبي إلى:
```
/Admin/Dashboard
```

## هيكل المشروع

```
├── Controllers/          → التحكم في صفحات العميل
├── Areas/Admin/           → لوحة تحكم الأدمن (Controllers + Views)
├── Models/                → نماذج البيانات (Book, Order, Cart...)
├── Data/                  → قاعدة البيانات والـ Seeding
├── Views/                 → صفحات العميل (HTML/Razor)
├── wwwroot/                → CSS والصور
└── Migrations/             → سجل تغييرات قاعدة البيانات
```

## ملاحظات مهمة

- تتبع الشحن في هذه النسخة **داخلي فقط** (تغيير الحالة يدويًا من لوحة التحكم)، بدون تكامل مع شركات شحن خارجية.
- الأدمن الحالي هو المستخدم الوحيد المصرح له بإدارة الكتب والطلبات.
- قاعدة البيانات المستخدمة هي SQLite، مناسبة للمشاريع الصغيرة والمتوسطة. في حالة الرغبة بالتوسع لاحقًا يمكن الترقية لـ SQL Server.

## للدعم الفني

في حالة مواجهة أي مشكلة أثناء التشغيل، تأكدي من:
1. تثبيت .NET SDK بالإصدار الصحيح
2. تنفيذ أمر `dotnet restore` قبل التشغيل
3. تنفيذ أمر `dotnet ef database update` قبل أول تشغيل
